# Inception-of-Things (IoT)

A System Administration exercise designed to teach Kubernetes deployment and management through hands-on practice.

## What is this?

This project is a comprehensive learning exercise that introduces students to modern container orchestration and deployment practices. It covers the complete journey from basic Kubernetes setup to advanced GitOps workflows.

## Purpose

The project aims to provide practical experience with:
- **Container orchestration** using Kubernetes
- **Infrastructure as Code** principles
- **Continuous deployment** methodologies
- **Service mesh** and networking concepts
- **DevOps** best practices
